# Project Brief — HMA (Htet Myat Aung)

## Overview
HMA is a web platform where an admin (Htet Myat Aung) manages and shares free Apple ID accounts with registered users. Users log in with admin-created credentials to view and copy Apple ID details. The site also includes a blog for iOS tutorials and tips.

## Core Requirements

### Primary Goal
Provide a clean, professional platform for sharing free Apple ID credentials with trusted users.

### Target Users
- **Admin (Htet Myat Aung)**: Manages Apple IDs, blog posts, users, and site settings
- **Registered Users**: Created by admin, can view/copy Apple ID credentials and read blog posts
- **Guests**: See login page only — no content access without authentication

### Key Features
1. **Apple ID Management** — Admin CRUD; users view/copy credentials
2. **User Management** — Admin creates accounts, can ban/unban
3. **Blog System** — iOS tutorials, tips, and guides
4. **Issue Reporting** — Users can report problems with Apple IDs
5. **Site Settings** — Alert banners, social links, branding
